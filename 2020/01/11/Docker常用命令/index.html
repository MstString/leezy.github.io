常见问题

在安装docker的虚拟机重启后，再次进入会发现docker 容器的状态为退出 docker ps -a，这个时候有两个选择

  1. 删除容器 docker rm <container ID>， 如果有容器中的数据很重要可以保存为images docker commit <container ID>
  2. 重启docker容器 docker start <container ID>
